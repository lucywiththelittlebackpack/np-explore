# National Parks Explorer

The world's #1 website for planning, finding, and exploring national parks around the world.

## Features
- **Global Park Directory**: Explore national parks from every continent.
- **Wildlife Focus**: Specialized tools for bird photographers and wildlife enthusiasts.
- **Accommodation**: Find the best places to stay near or inside the parks.
- **Premium Experience**: Immersive design to inspire your next adventure.

## Tech Stack
- **Framework**: Next.js (App Router)
- **Language**: TypeScript
- **Styling**: Vanilla CSS (Modules)

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.
